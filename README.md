# API RESTful com FastAPI e Python

Este projeto consiste no desenvolvimento de uma API RESTful completa utilizando Python e FastAPI, com foco em arquitetura backend moderna, autenticação segura, organização modular e boas práticas de desenvolvimento.

A aplicação foi estruturada para simular um sistema real de gerenciamento de pedidos, implementando funcionalidades de autenticação, autorização, manipulação de banco de dados e controle de acesso por usuário.

---

# 🚀 Tecnologias Utilizadas

- Python
- FastAPI
- SQLAlchemy
- Alembic
- PostgreSQL
- Pydantic
- JWT Authentication
- OAuth2
- bcrypt
- Git e GitHub

---

# 📌 Funcionalidades

- Cadastro de usuários
- Sistema de autenticação JWT
- Proteção de rotas
- Controle de permissões
- CRUD de usuários
- CRUD de pedidos
- CRUD de itens
- Versionamento de banco de dados
- Validação de dados com Pydantic
- Documentação automática Swagger/OpenAPI

---

# 🧠 Conceitos Aplicados

- APIs RESTful
- Arquitetura modular
- Autenticação e autorização
- ORM com SQLAlchemy
- Migrations com Alembic
- Relationship e Lazy Loading
- Response Models
- Boas práticas backend
- Segurança com bcrypt e JWT

---

# 🎯 Objetivo do Projeto

O objetivo deste projeto foi aprofundar conhecimentos em desenvolvimento backend com Python, aplicando conceitos modernos de construção de APIs seguras, organizadas e escaláveis, simulando padrões utilizados em aplicações reais.
